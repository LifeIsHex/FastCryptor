# Fast Cryptor #

* What is Fast Cryptor?

    Fast Cryptor is a file encryptor\decryptor for windows x64 and x32 architecture. Designed to be simple and user-friendly, yet powerful and portable. It uses AES 256 EAX algorithm.
  
* Note:
  <b> Fast Cryptor comes with absolutely no guarantee. Use at your own risk. </b>


## Features ##

  * Drag and drop support.
  * Unicode support.
  * AES 256 EAX encryption\decryption algorithm.
  * Secure delete source file.
  * Portable and light.
